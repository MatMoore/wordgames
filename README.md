# Word games

A thing to generate cards with random words on them from a list.

Based on [Random Noun](https://kevan.org/games/randomnoun) by Kevan Davis.

## Local development

### Dependencies

- Install yarn
- `yarn install`

### Running

- `yarn build`
- `yarn serve`

### Run the tests

- `yarn test`
